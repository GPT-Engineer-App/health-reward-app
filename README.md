# health-reward-app

a healthcare ios app that rewards people with money for doing physical activities, eating healthy and staying healthy; we connect with HSA and FSA for savings, spending, credit and investment or they can choose to spend the cash

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/health-reward-app.git
cd health-reward-app
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
